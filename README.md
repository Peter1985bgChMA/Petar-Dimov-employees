# Petar-Dimov-employees
Team Longest Period Together App


**Description**

A C#/.NET application that finds the pair of employees who have worked together for the longest overlapping period on the same project.
The input data is read from a CSV file and overlapping work periods are calculated to determine the longest shared duration.


**Technologies**

  - C#
  - Blazor Web Assembly


**Input Format**

CSV file with the following columns:
EmployeeID, EmployeeName, ProjectID, DateFrom, DateTo

  - DateTo may be NULL (treated as the current date)


**How to Run**

1. Open TeamLongestPeriod.sln in Visual Studio
2. Build the solution
3. Run the project


**Notes**

Build artifacts (bin, obj, .vs) are excluded
Repository structure follows standard .NET conventions
